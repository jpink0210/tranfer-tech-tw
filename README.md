# 前言：

- 這邊是用 node 自己在 local 建設

- 不想要用 gitbook 官方釋出的，Web Gui 做綁定
只是一個紀錄歷程不用那麼浩大。


# 安裝和啟動

```
確認一下你本機的 node -v, 尤其是每次終端機重開
nvm install v12.16.3 / nvm use v12.16.3

npm install -g gitbook-cli --save-dev

gitbook init

gitbook serve
or
gitbook serve --port 8081

```
